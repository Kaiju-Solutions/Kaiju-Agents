Controllers

Controllers allow for easily access agent callbacks without having to worry about binding to C# events. Two versions are provided of which you can extend from:

  1. KaijuController - This attaches directly to an agent on the same GameObject and listens to callbacks specifically for the associated agent.
  2. KaijuGlobalController - This controller receives all global callbacks for all agents in the scene. Unlike the KaijuController, this does not need to be attached to an agent. In most use cases, a KaijuController will be more effective to use than a KaijuGlobalController.
