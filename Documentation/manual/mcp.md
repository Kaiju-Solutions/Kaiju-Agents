# Model Context Protocol (MCP)

Kaiju Agents integrates with [Coplay](https://www.coplay.dev "Coplay")'s [Unity MCP](https://github.com/CoplayDev/unity-mcp "Unity MCP"), adding in tools to speed up Kaiju Agents workflows. These tools allow for speeding up the ability of the connected MCP clients to create agents and basic scenes for using them.
