# Behavior Graph

When [Unity Behavior](https://docs.unity3d.com/Packages/com.unity.behavior@latest "Unity Behavior") is installed in the project, nodes to allow for all types of [movement](https://agents.kaijusolutions.ca/manual/movement.html "Movement") and using [sensors](https://agents.kaijusolutions.ca/manual/sensors.html "Sensors") and [actuators](https://agents.kaijusolutions.ca/manual/actuators.html "Actuators") are available. Additionally, nodes have been made for the ["Microbes"](https://agents.kaijusolutions.ca/manual/microbes.html "Microbes") and ["Capture the Flag"](https://agents.kaijusolutions.ca/manual/capture-the-flag.html "Capture the Flag") exercises. To get started, follow the [Unity Behavior](https://docs.unity3d.com/Packages/com.unity.behavior@latest "Unity Behavior") documentation.

- [Core API.](https://agents.kaijusolutions.ca/api/KaijuSolutions.Agents.Behavior.html "Behavior - Core")
- [Movement API.](https://agents.kaijusolutions.ca/api/KaijuSolutions.Agents.Behavior.Movement.html "Behavior - Movement")
- [Sensors API.](https://agents.kaijusolutions.ca/api/KaijuSolutions.Agents.Behavior.Sensors.html "Behavior - Sensors")
- [Actuators API.](https://agents.kaijusolutions.ca/api/KaijuSolutions.Agents.Behavior.Actuators.html "Behavior - Actuators")
- [Actuators API.](https://agents.kaijusolutions.ca/api/KaijuSolutions.Agents.Behavior.Actuators.html "Behavior - Actuators")
- [Microbes API.](https://agents.kaijusolutions.ca/api/KaijuSolutions.Agents.Behavior.Exercises.Microbes.html "Behavior - Microbes")
- [Capture the Flag API.](https://agents.kaijusolutions.ca/api/KaijuSolutions.Agents.Behavior.Exercises.CTF.html "Behavior - Microbes")
