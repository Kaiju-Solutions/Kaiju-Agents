# Samples and Exercises

[Samples](#samples "Samples") provide quick learning resources for Kaiju Agents, and [exercises](#exercises "Exercises") provide an easy way to build your skills with Kaiju Agents. To access [samples](#samples "Samples") and [exercises](#exercises "Exercises"), from the top menu, go to `Window > Package Management > Package Manager`, go to the `In Project` tab, select `Kaiju Agents`, and click on `Samples`. From there, click the `Import` button next to the [sample](#sample "Samples") or [exercise](#exercises "Exercises") you want. **Note that for all [samples](#sample "Samples") and [exercises](#exercises "Exercises"), you must first import the [`Materials` sample](#materials "Materials Sample").**

## Samples

### Materials

The `Materials` sample simply contains [materials](https://docs.unity3d.com/Manual/class-Material.html "Materials") used throughout the other samples and [exercises](#exercises "Exercises"). If materials appear pink, from the top menu, go to `Window > Rendering > Render Pipeline Converter`, choose `Built-in to URP` or `Built-in to HDRP`, select all materials, click `Initialize Converters`, and lastly click `Convert Assets`.

### Movement

This sample provides multiple scenes to demonstrate [movements](https://agents.kaijusolutions.ca/manual/movement.html "Movement"). There are scenes for:

- Comparing [seek](https://agents.kaijusolutions.ca/manual/movement.html#seek "Seek") and [pursue](https://agents.kaijusolutions.ca/manual/movement.html#pursue "Pursue").
- Comparing [flee](https://agents.kaijusolutions.ca/manual/movement.html#flee "Flee") and [evade](https://agents.kaijusolutions.ca/manual/movement.html#evade "Evade").
- Demonstrating [wandering](https://agents.kaijusolutions.ca/manual/movement.html#wander "Wander").
- Demonstrating [sepration](https://agents.kaijusolutions.ca/manual/movement.html#sepration "Separation").
- Demonstrating [obstacle avoidance](https://agents.kaijusolutions.ca/manual/movement.html#obstacle-avoidance "Obstacle Avoidance").
- Demonstrating [path following](https://agents.kaijusolutions.ca/manual/movement.html#path-following "Path Following").

### Keyboard

This sample provides a scene with an agent that can be controlled with a keyboard. To control the agent, use WASD or the arrow keys to move, and Q and E to turn.

### Box Destroyer

This provides a sample scene of an agent which is tasked with destroying "boxes". The agent demonstrates the use of an [everything vision sensor](https://agents.kaijusolutions.ca/manual/sensors.html#everything-vision-sensor "Everything Vision Sensor") and an [everything attack actuator](https://agents.kaijusolutions.ca/manual/actuator.html#everything-attack-actuator "Everything Attack Actuator") being controlled by a [controller](https://agents.kaijusolutions.ca/manual/controllers.html "Controllers").

## Exercises

Several exercises tailored to help you learn Kaiju Agents or use in an education environment are provided. Each exercise is more complicated than the last, providing an efficient learning experience for Kaiju Agents.

1. [Cleaner](https://agents.kaijusolutions.ca/manual/cleaner.html "Cleaner")
2. [Microbes](https://agents.kaijusolutions.ca/manual/microbes.html "Microbes")
3. [Capture the Flag](https://agents.kaijusolutions.ca/manual/capture-the-flag.html "Capture the Flag")
