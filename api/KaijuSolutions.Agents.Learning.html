Table of Contents

Namespace KaijuSolutions.Agents.Learning

Integration for Unity ML-Agents.

Classes

KaijuLearningActuator

Have any generic KaijuActuator work with ML-Agents. This has three discrete actions, being to do nothing, run the KaijuActuator, or cancel running it. These actions are masked accordingly. This has no Heuristic(in ActionBuffers) implementation, and you will need to extend this class and override for your use case.

KaijuLearningActuatorBase

Base class for integrating all actuators with ML-Agents.

KaijuLearningAgentActuator

Actuator for controlling an KaijuAgent for ML-Agents.

KaijuLearningAgentSensor

Sensor for getting information from an KaijuAgent for ML-Agents.

KaijuLearningAttackActuator

Use KaijuAttackActuators with ML-Agents. This has three discrete actions, being to do nothing, run the KaijuActuator, or cancel running it. These actions are masked accordingly. Compared to the KaijuLearningActuator, this takes into account the KaijuAttackActuator being on cooldown when masking actions. This has no Heuristic(in ActionBuffers) implementation, and you will need to extend this class and override for your use case.

KaijuLearningCastSensor

Integrate KaijuCastSensors with ML-Agents. This normalizes all rays from zero to one.

KaijuLearningControlActuator

Actuator for controlling the movement of an KaijuAgent via the Control property for ML-Agents. This has two continuous actions which are assigned to the Control property. This has no Heuristic(in ActionBuffers) implementation, and you will need to extend this class and override for your use case.

KaijuLearningRotationSensor

Sensor for getting the OrientationY of an KaijuAgent for ML-Agents normalized between zero and one.

KaijuLearningSensor

Base class for integrating all sensors with ML-Agents.

KaijuLearningSpinActuator

Actuator for controlling the rotation of an KaijuAgent via the Spin property for ML-Agents. This has one continuous action which is assigned to the Spin property. This has no Heuristic(in ActionBuffers) implementation, and you will need to extend this class and override for your use case.

KaijuLearningVelocitySensor

Sensor for getting the normalized Velocity of an KaijuAgent for ML-Agents.

KaijuLearningVisionSensor

Integrate KaijuVisionSensor<T>s with ML-Agents. All positions are normalized from negative one to positive one based on detection distance.
