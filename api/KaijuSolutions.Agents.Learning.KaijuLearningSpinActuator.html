    Class KaijuLearningSpinActuator

    Actuator for controlling the rotation of an KaijuAgent via the Spin property for ML-Agents. This has one continuous action which is assigned to the Spin property. This has no Heuristic(in ActionBuffers) implementation, and you will need to extend this class and override for your use case.
